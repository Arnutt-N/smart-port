**handleDiverse(PDO $pdo, string $method, array $path): void**
Switch on $method dispatching to sub-functions. Same pattern as probation.php for GET (list vs detail), POST, PUT, DELETE.

**getDiverseList(PDO $pdo): void** (per D-02, D-03)
- Read `$_GET['personnel_id']`, `$_GET['limit']` (default 20), `$_GET['offset']` (default 0)
- Build WHERE clause: `WHERE de.personnel_id = ?` if personnel_id provided
- SELECT query joining personnel: `SELECT de.*, CONCAT(p.first_name, ' ', p.last_name) AS full_name FROM diverse_experience de LEFT JOIN personnel p ON de.personnel_id = p.personnel_id`
- Include ALL columns from table: experience_id, personnel_id, from_job_series, from_work_group, from_division, from_org_id, from_province, from_start_date, from_end_date, from_total_days, to_job_series, to_work_group, to_division, to_org_id, to_province, to_start_date, to_end_date, to_total_days, is_diff_job_series, is_diff_org, is_diff_location, is_diff_work_nature, diff_count, qualified_date, created_at
- ORDER BY de.created_at DESC, LIMIT/OFFSET
- Count query for pagination
- Loop adding Thai dates: from_start_date_thai, from_end_date_thai, to_start_date_thai, to_end_date_thai, qualified_date_thai
- Response: `{ "success": true, "data": [...], "pagination": { ... } }`

**getDiverseDetail(PDO $pdo, int $id): void**
- SELECT with JOIN to personnel, WHERE `de.experience_id = ?`
- 404 if not found: `['error' => 'ไม่พบรายการนับแตกต่าง']` (per D-15)
- Add Thai date fields
- Response: `{ "success": true, "data": {record} }`

**createDiverse(PDO $pdo): void** (per D-08, D-09, D-10)
- Read JSON body
- Required fields: `['personnel_id']` (minimum — the rest are optional per the schema)
- Validation loop with Thai error: `"กรุณาระบุ {$field}"`
- Compute from_total_days server-side (per D-10): if from_start_date and from_end_date both present, `(new DateTime($data['from_end_date']))->diff(new DateTime($data['from_start_date']))->days + 1`, else NULL
- Compute to_total_days server-side (per D-10): same pattern with to_start_date and to_end_date
- Compute qualified_date in PHP (per D-09): sum the 4 booleans `$diffCount = intval($data['is_diff_job_series'] ?? 0) + intval($data['is_diff_org'] ?? 0) + intval($data['is_diff_location'] ?? 0) + intval($data['is_diff_work_nature'] ?? 0)`. If `$diffCount >= 3` and `to_start_date` is not empty, set `$qualifiedDate = $data['to_start_date']`, else NULL.
- INSERT columns (per D-08 — NO diff_count): personnel_id, from_job_series, from_work_group, from_division, from_org_id, from_province, from_start_date, from_end_date, from_total_days, to_job_series, to_work_group, to_division, to_org_id, to_province, to_start_date, to_end_date, to_total_days, is_diff_job_series, is_diff_org, is_diff_location, is_diff_work_nature, qualified_date
- Use 22 `?` placeholders matching the 22 columns above
- Return 201: `{ "success": true, "experience_id": N }` (per D-16)

**updateDiverse(PDO $pdo, int $id): void**
- Read JSON body
- Verify record exists, return 404 with `'ไม่พบรายการนับแตกต่าง'` if not found
- Allowed fields (per D-08 — NO diff_count): `['from_job_series', 'from_work_group', 'from_division', 'from_org_id', 'from_province', 'from_start_date', 'from_end_date', 'to_job_series', 'to_work_group', 'to_division', 'to_org_id', 'to_province', 'to_start_date', 'to_end_date', 'is_diff_job_series', 'is_diff_org', 'is_diff_location', 'is_diff_work_nature']`
- Build SET clause using allowed-fields whitelist
- If any date field (from_start_date, from_end_date, to_start_date, to_end_date) or any boolean flag changes: fetch current record, merge with new values, recompute from_total_days, to_total_days, and qualified_date. Add these to SET clause.
- Execute UPDATE, return `{ "success": true }`

**deleteDiverse(PDO $pdo, int $id): void**
- DELETE FROM diverse_experience WHERE experience_id = ?
- Check rowCount(), 404 if 0 with `'ไม่พบรายการนับแตกต่าง'`
- Return `{ "success": true }`
