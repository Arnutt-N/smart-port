Imports:
```javascript
import { ref, watch, onMounted } from 'vue'

import { Users, CheckCircle, Clock, AlertTriangle, Search, RefreshCw } from 'lucide-vue-next'
```

Reactive state:
```javascript
const { fetchList } = useProbation()
const search = ref('')
const searchTimeout = ref(null)
const loading = ref(false)
const error = ref(null)
const enrollments = ref([])
const summary = ref({ total: 0, in_progress: 0, near_deadline: 0, overdue: 0 })
const pagination = ref({ total: 0, limit: 20, offset: 0, has_more: false })
```

**Functions:**

`loadData()`:
- Sets `loading = true`, `error = null`
- try: calls `fetchList({ search: search.value, limit: pagination.value.limit, offset: pagination.value.offset })`
- Updates `enrollments`, `summary`, `pagination` from result
- catch: sets `error = err.message || 'เกิดข้อผิดพลาดในการโหลดข้อมูล'`
- finally: `loading = false`

`handleSearch()` -- Debounced search (300ms, per pitfall 3):
- `clearTimeout(searchTimeout.value)`
- `searchTimeout.value = setTimeout(() => { pagination.value.offset = 0; loadData() }, 300)`

`handleOffsetChange(newOffset)`:
- `pagination.value.offset = newOffset`
- `loadData()`

`onMounted(() => loadData())`

     - "ทั้งหมด" -- value: `summary.total`, icon: Users, default blue colors
     - "กำลังดำเนินการ" -- value: `summary.in_progress`, icon: CheckCircle, iconBgClass: "bg-green-50", iconClass: "text-green-500"
     - "ใกล้ครบกำหนด" -- value: `summary.near_deadline`, icon: Clock, iconBgClass: "bg-yellow-50", iconClass: "text-yellow-500"
     - "เกินกำหนด" -- value: `summary.overdue`, icon: AlertTriangle, iconBgClass: "bg-red-50", iconClass: "text-red-500"

3. **Search bar** (per D-20, PT-10): search input with:
   - `v-model="search"` and `@input="handleSearch"`
   - placeholder: "ค้นหาชื่อ-สกุล, ตำแหน่ง หรือหน่วยงาน..."
   - class: "input flex-1 max-w-md" with Search icon inside

4. **Error state** (per D-16): `<EmptyState v-if="error && !loading">`
   - icon: AlertTriangle
   - title: "เกิดข้อผิดพลาด"
   - description: error message
   - Slot: `<button @click="loadData" class="mt-3 px-4 py-2 bg-blue-500 text-white rounded-lg hover:bg-blue-600 text-sm">ลองใหม่</button>`

5. **Loading state** (per D-15): `<SkeletonLoader v-else-if="loading" type="table" :rows="8" />`

6. **Data table** (per PT-07): white rounded-lg shadow-sm border. Columns:

   - ชื่อ-สกุล -- `e.name` (font-medium text-gray-900)
   - ตำแหน่ง -- `e.position`
   - หน่วยงาน -- `e.department`
   - วันเริ่มทดลอง -- `e.startDate`
   - วันครบกำหนด -- `e.endDate`
   - วันคงเหลือ -- `e.remainingDays` + " วัน" with `:class="getRemainingDaysClass(e.remainingDays)"` (per PT-08, D-12)

7. **Empty state** when `enrollments.length === 0 && !loading && !error`: EmptyState "ไม่พบข้อมูล"

8. **Pagination** (per D-18, D-21): `<PaginationBar :total="pagination.total" :limit="pagination.limit" :offset="pagination.offset" @update:offset="handleOffsetChange" />`
