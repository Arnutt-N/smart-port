**Functions:**

`fetchData()`:
- Set `loading = true`, `error = null`
- Call `fetchList({ search: searchQuery.value, limit: pagination.value.limit, offset: pagination.value.offset })`
- On success: update `rows`, `summary`, `pagination`
- On error: set `error` to error message
- Set `loading = false`

`onSearchInput()`:
- `clearTimeout(searchTimeout)`
- `searchTimeout = setTimeout(() => { pagination.value.offset = 0; fetchData() }, 300)`
- (Per Pitfall 4: reset offset on search change)

**onMounted:** call `fetchData()`.
