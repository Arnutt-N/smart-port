**Key functions:**

`loadCandidates()` -- Fetches data for current sub-tab level:
- Sets `loading = true`, `error = null`
- try: calls `fetchByLevel(currentLevel.value, { search: search.value, limit: pagination.value.limit, offset: pagination.value.offset })`
- Updates `candidates`, `summary`, `pagination` from result
- catch: sets `error = err.message || 'เกิดข้อผิดพลาด'`
- finally: `loading = false`

`loadOverview()` -- Fetches all levels with Promise.allSettled (per D-08, pitfall 2):
- Sets `overviewLoading = true`, `overviewError = null`
- Calls `Promise.allSettled(allLevels.map(level => fetchByLevel(level, { limit: 1000 })))`
- Processes results:
  - For fulfilled: accumulate summary counts and data arrays
  - `generalCount` = sum of totals for O2+O3 results
  - `academicCount` = sum of totals for K2+K3+K4 results
  - `totalQualified` = sum of all `summary.qualified`
  - `totalNotYet` = sum of all `summary.not_yet`
  - `totalCheckData` = sum of all `summary.check_data`
  - `top5` = concatenate all data arrays, sort by `remainingDays` ascending (null last), take first 5
- Sets `overviewData` with computed aggregates
- catch: sets `overviewError`
- finally: `overviewLoading = false`

`handleSearch()` -- Debounced search (300ms, per pitfall 3):
- Clears existing timeout
- Sets new 300ms timeout that resets `pagination.value.offset = 0` (per pitfall 4) and calls `loadCandidates()`

`handleOffsetChange(newOffset)` -- Pagination handler:
- Sets `pagination.value.offset = newOffset`
- Calls `loadCandidates()`

**Watchers:**

Watch `() => props.section` -- when section changes:
- Reset search, offset to 0
- If section has subTabs, set `activeSubTab` to first sub-tab's level (per D-03)
- If overview, call `loadOverview()`
- If placeholder, do nothing
- If has sub-tabs, call `loadCandidates()`

Watch `activeSubTab` -- when sub-tab changes (per D-17):
- Reset `pagination.value.offset = 0` and `search.value = ''`
- Call `loadCandidates()`

`onMounted()`:
- Initialize `activeSubTab` based on current section (first sub-tab level if applicable)
- If overview, call `loadOverview()`
- If has sub-tabs, call `loadCandidates()`
