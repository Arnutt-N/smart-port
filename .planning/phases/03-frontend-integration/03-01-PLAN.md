**frontend/src/composables/useCandidates.js** (per D-22, D-24):
- Export function `useCandidates()` that calls `useApi()` internally
- Method `fetchByLevel(targetLevel, { search = '', limit = 20, offset = 0 } = {})`:
  - Builds URLSearchParams with search, limit, offset
  - Calls `api.get('/candidates/${targetLevel}?${params}')`
  - Maps each row from snake_case to camelCase via `mapCandidateRow()`
  - Returns `{ success, data, summary, pagination }`
- Method `fetchDetail(targetLevel, personnelId)`:
  - Calls `api.get('/candidates/${targetLevel}/${personnelId}')`
  - Returns mapped result
- Internal `mapCandidateRow(row)` mapping:
  - `row.personnel_id` -> `personnelId`
  - `row.full_name` -> `name`
  - `row.current_position` -> `currentPosition`
  - `row.current_level_code` -> `currentLevelCode`
  - `row.current_level_name` -> `currentLevelName`
  - `row.level_start_date_thai` -> `levelStartDate`
  - `row.qualification_date_thai` -> `qualificationDate`
  - `row.remaining_days` -> `remainingDays`
  - `row.status` -> `status`
  - `row.department` -> `department`
- Export `getRemainingDaysClass(days)` as standalone named export (shared by both pages, per D-12):
  - `days === null || days === undefined` -> `'text-gray-400'`
  - `days < 7` -> `'text-red-600 font-medium'` (includes negative values)
  - `days <= 14` -> `'text-orange-600'`
  - `days <= 30` -> `'text-yellow-600'`
  - else -> `'text-green-600'`
