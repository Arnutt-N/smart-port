# แผนจัดทำเอกสารโครงการ Smart Port (HRIS)

> ระบบบริหารงานบุคคล — สำนักงานปลัดกระทรวงยุติธรรม
> วันที่จัดทำ: 26 มีนาคม 2569

---

## ภาพรวม

เอกสารชุดนี้เป็นแผนการจัดทำเอกสารประกอบโครงการระบบ Smart Port
เพื่อใช้รายงานหัวหน้างานและผู้อำนวยการ รวมถึงเป็นเอกสารอ้างอิงของโครงการ

---

## กลุ่ม 1: เอกสารสำหรับผู้อำนวยการ / ผู้บริหารระดับสูง

| # | ชื่อเอกสาร | รูปแบบ | จุดประสงค์ |
|---|-----------|--------|-----------|
| 1 | **รายงานสรุปผู้บริหาร (Executive Summary)** | PDF / HTML Presentation | ภาพรวมระบบ ประโยชน์ สถานะ ในหน้าเดียวหรือ 3-5 สไลด์ |
| 2 | **ข้อเสนอโครงการ (Project Proposal)** | PDF | เหตุผลความจำเป็น งบประมาณ ระยะเวลา ผลที่คาดว่าจะได้รับ |
| 3 | **รายงานความก้าวหน้า (Progress Report)** | PDF | สถานะงานปัจจุบัน % ที่เสร็จ ปัญหาอุปสรรค แผนถัดไป |
| 4 | **Presentation สรุประบบ** | HTML Slides | สไลด์นำเสนอพร้อม screenshot ของระบบจริง |

## รูปแบบไฟล์ที่จัดทำได้

| รูปแบบ | คำอธิบาย |
|--------|---------|
| **Markdown (.md)** | ไฟล์ต้นฉบับ แก้ไขง่าย แปลงเป็น PDF ได้ |
| **HTML Report** | เอกสารที่ print เป็น PDF ได้สวยงาม มีสารบัญ |
| **HTML Presentation** | สไลด์นำเสนอเปิดในเบราว์เซอร์ พร้อม animation |

## หมายเหตุ

- เอกสารทุกฉบับจะอ้างอิงข้อมูลจริงจาก source code ของระบบ Smart Port

- รูปภาพประกอบ (screenshot, diagram) จะจัดเก็บใน `docs/project-documents/images/`
